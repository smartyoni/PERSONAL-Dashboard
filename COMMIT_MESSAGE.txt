Web Share Target API 구현 (모바일 공유 기능)

## 개요
모바일 시스템 공유 메뉴에서 "최영현보드"를 선택하여 텍스트를 공유할 수 있도록 Web Share Target API를 구현했습니다.

## 변경 사항

### 1. public/manifest.json
- share_target 설정 추가
- action: /PERSONAL-Dashboard/share-target
- params: title, text, url 매핑

### 2. public/sw.js (새로 생성)
- Web Share Target API를 위한 Service Worker
- POST 요청 감지 및 처리
- FormData에서 title, text, url 추출
- URL 파라미터로 변환하여 리다이렉트

### 3. index.html
- 기존 Service Worker 제거 코드 → 등록 코드로 변경
- /PERSONAL-Dashboard/sw.js 등록
- 오류 처리 및 로깅 추가

### 4. App.tsx
- ListItem 타입 import 추가
- URL 파라미터 처리 useEffect 추가
- shared=true&text=... 파라미터 감지
- IN-BOX에 항목 자동 추가
- 메인 탭으로 자동 전환
- IN-BOX 섹션으로 자동 스크롤
- 3초간 황색 하이라이트 표시

## 기술 사항

- Service Worker는 load 이벤트 후 등록하여 초기 로딩 성능 영향 없음
- 공유 데이터는 title\ntext\nurl 형식으로 합쳐짐
- URL 파라미터는 history.replaceState()로 자동 제거
- 모든 경로에 베이스 경로 /PERSONAL-Dashboard/ 포함

## 테스트 방법

로컬 테스트:
```bash
npm run dev
# http://localhost:3000/PERSONAL-Dashboard/?shared=true&text=테스트
```

프로덕션 빌드:
```bash
npm run build
npm run preview
```

모바일 테스트:
```bash
npm run deploy
# HTTPS URL에서 PWA 설치 후 시스템 공유 테스트
```
