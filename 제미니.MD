# 배포 정보

배포는 GitHub Pages를 사용하여 진행합니다.

## 배포 방법
사용자 레포지토리에는 GitHub Actions 워크플로우(`deploy.yml`)가 설정되어 있어, 코드를 `main` 브랜치에 푸시하면 자동으로 빌드와 배포가 진행됩니다.

1. **로컬 테스트 후 변경사항 커밋 및 푸시**: 변경 사항을 모두 커밋한 뒤 `git push origin main` 명령어를 실행합니다.
2. **자동 배포 확인**: 코드가 푸시되면 [GitHub 레포지토리의 Actions 탭](https://github.com/smartyoni/PERSONAL-Dashboard/actions)에서 `Deploy to GitHub Pages` 액션이 자동으로 실행되어 반영됩니다.

- 호스팅 URL: https://smartyoni.github.io/PERSONAL-Dashboard/
