# 프로젝트 업데이트 로그 (제미니)

## 배포 지침
- 배포는 GitHub Pages를 통해 자동으로 진행됩니다.
- 작업 완료 후 `git push origin main`을 수행하면 GitHub Actions가 배포를 시작합니다.

## 최근 작업 내역

### 2026-03-16
1. **즐겨찾기 메모 저장 오류 수정**
   - 즐겨찾기 섹션에서 메모 수정 시 현재 활성화된 탭이 아닌, 실제 항목이 속한 탭(`tabId`)에 저장되도록 수정 완료.
2. **모바일 플로팅 버튼(FAB) 개선**
   - 전체 목차: 🗺️ -> **전체** (텍스트 변경)
   - 현재 섹션: 📋 -> **현재** (텍스트 변경)
   - 북마크 토글 기능 제거 및 **인박스(IN-BOX)** 바로가기 버튼 추가.
   - 텍스트 가독성을 위해 `text-sm font-bold` 스타일 적용.
3. **모바일 뒤로가기 버튼 연동**
   - 브라우저/스마트폰 자체 뒤로가기 버튼 클릭 시 앱이 종료되지 않고 현재 열린 모달(메모, 지도, 태그 등)이 먼저 닫히도록 History API 연동.
4. **메모 모달 리치 텍스트 에디터 및 구분선 개선**
   - 기존 `textarea`를 `contentEditable` 기반의 리치 텍스트 에디터로 업그레이드.
   - **편집 모드에서도 실시간 구분선 확인**: 이제 편집 중에도 `---divider---` 텍스트 대신 실제 파란색 구분선이 즉시 나타납니다.
   - 콜론(`:`) 및 구분선(`―`) 기호 삽입 시 커서 위치를 정확히 유지하도록 Selection API 연동.
